清理未使用的安卓镜像

安卓容器的镜像是Docker进行统一管理的。使用ssh连接盒子的Linux，默认用户名 user 密码 myt 在sudo su输入密码切换到root用户后进行操作。 使用

docker images

查看本地镜像列表。

使用

docker image prune -a

清理所有未使用的镜像

更多Docker命令和指南可以自行网上搜索或询问GPT等。

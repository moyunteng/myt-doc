Q1ubuntu刷机教程​

  1. 点击固件，选择Q1的ubuntu固件
  2. 选择后在此处显示固件路径
  3. 点击升级进行刷机

此处可参考C1 ubuntu固件的刷写过程

  1. 安装驱动（首次需要）

  2. 启动刷写工具，如本教程上图，选择“升级固件”

  3. 按住Q1按住maskrom按钮，通电，进入刷机模式

    此时刷机工具会显示
